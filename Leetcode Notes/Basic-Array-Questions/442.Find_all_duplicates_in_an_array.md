## 442. Find All Duplicates in an Array

<h3>

```cpp
class Solution {
public:
    vector<int> findDuplicates(vector<int>& nums) {
       vector<int> ans;

       // as the number is ranging from 1 to n
       // we can use elements inside the array to iterate
       //    approach is pretty simple : iterate through elements inside arrays
       // and mark them negative : if a negative element found while traversing that is it's already seen , push it to ans
       // and we are good to go

       for(int i=0; i<nums.size(); i++)
       {
            int index = abs(nums[i])-1;
            if(nums[index] <0) ans.push_back(abs(nums[i]));
            else nums[index] *=-1;
       }

        return ans;
    }
};
```
